## Summary

Criar tabela Delta `observer.diagnostics` que registra cada diagnostico do Observer com metricas completas, e um dashboard Databricks SQL para acompanhar performance do agente ao longo do tempo.

## Context

Hoje o Observer loga diagnosticos apenas no output do notebook. Nao ha historico persistente, metricas de custo, nem visibilidade sobre a eficacia dos diagnosticos. Precisamos de dados estruturados para medir e melhorar o agente.

## User Story

Como gestor do pipeline, quero um dashboard mostrando quantos diagnosticos o Observer fez, custo em tokens, taxa de sucesso dos PRs, e tempo medio de resolucao, para avaliar o ROI do agente autonomo.

## Acceptance Criteria

- [ ] Tabela `observer.diagnostics` existe no Unity Catalog com schema documentado
- [ ] Todo diagnostico (sucesso ou falha) e registrado na tabela
- [ ] Metricas incluem: timestamp, provider, modelo, tokens (in/out), confianca, PR URL, custo estimado
- [ ] Dashboard SQL com pelo menos 4 paineis (diagnosticos/dia, custo acumulado, confianca media, top erros)
- [ ] Dados acessiveis via SQL para queries ad-hoc

## Dependencies

Nenhuma — pode ser implementada independentemente.

## Out of Scope

## Technical Notes

### Schema da Tabela

```sql
CREATE TABLE observer.diagnostics (
  id STRING,                    -- UUID
  timestamp TIMESTAMP,
  job_id BIGINT,
  job_name STRING,
  run_id BIGINT,
  failed_task STRING,
  error_message STRING,
  error_hash STRING,            -- SHA-256 do error_message (para dedup futura)
  diagnosis STRING,
  root_cause STRING,
  fix_description STRING,
  file_to_fix STRING,
  confidence DOUBLE,
  requires_human_review BOOLEAN,
  pr_url STRING,
  pr_number INT,
  branch_name STRING,
  provider STRING,              -- "anthropic", "openai"
  model STRING,                 -- "claude-opus-4-20250514", "gpt-4o"
  input_tokens INT,
  output_tokens INT,
  estimated_cost_usd DOUBLE,    -- Calculado com base nos tokens
  duration_seconds DOUBLE,
  status STRING                 -- "success", "failed", "no_fix_proposed"
)
USING DELTA
```
