# Specification: Platform Frontend (Nuxt 4)

**Track ID:** platform-frontend_20260408
**Type:** Feature
**Created:** 2026-04-08
**Status:** Draft

## Summary

Frontend Nuxt 4.4.2 da plataforma conversacional. Interface estilo Claude Projects com sidebar de pipelines/conversas, chat SSE, settings de empresa com credenciais e QR Code WhatsApp, e gestao de usuarios (IAM). Atomic Design para componentes.

## User Story

Como usuario da plataforma, quero uma interface de chat intuitiva onde seleciono meu pipeline, converso com o agente, vejo acoes executadas (PRs, runs), e configuro credenciais da minha empresa.

## Acceptance Criteria

### Auth
- [ ] Login (email/senha), OAuth opcional (Google/GitHub)
- [ ] Middleware global de auth (redirect para /login)
- [ ] Role-based UI: viewer ve menos opcoes que admin

### Sidebar (estilo Claude Projects)
- [ ] Lista de pipelines com badge de status (SUCCESS/FAILED/RUNNING)
- [ ] Clique expande → lista de conversas do usuario
- [ ] Cada conversa: UUID, data, preview, icone do canal de origem
- [ ] Conversas de WhatsApp/Discord/Telegram aparecem aqui (com icone)
- [ ] "+ Nova conversa", deletar, buscar

### Chat
- [ ] SSE streaming de respostas do LLM
- [ ] Input com Shift+Enter, drag & drop de imagens/arquivos
- [ ] Renderizacao Markdown, code blocks (Shiki), tabelas
- [ ] Action cards: PR criado, run disparado, query executada
- [ ] Confirmacao inline para acoes perigosas
- [ ] "Agente pensando..." loading indicator

### Settings (root/admin only)
- [ ] Pagina /settings com tabs: Geral, Credenciais, Canais, Usuarios
- [ ] Tab Credenciais: Anthropic Key, Databricks, GitHub (input + "Testar conexao")
- [ ] Tab Canais: Discord token, Telegram token, WhatsApp QR Code
- [ ] WhatsApp: botao "Conectar" → mostra QR Code via WebSocket → status de conexao
- [ ] Tab Usuarios: lista, criar, editar role, desativar
- [ ] Feedback visual: "Conexao OK", "Chave invalida", loading states

### Atomic Design
- [ ] atoms/: AppButton, AppInput, AppBadge, AppAvatar, AppSpinner, ChannelIcon
- [ ] molecules/: StatusBadge, MessageBubble, ActionCard, SearchBar, FormField, ConfirmDialog
- [ ] organisms/: ChatWindow, SidebarNav, ThreadList, PipelineCard, SettingsForm, UserTable, QrCodePairing
- [ ] templates/: ChatLayout, AuthLayout, SettingsLayout, AdminLayout
