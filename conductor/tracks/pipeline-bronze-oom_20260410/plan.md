## Overview

Substituir o fluxo `boto3 -> pandas -> spark.createDataFrame` do `S3Lake.read_parquet` por leitura nativa via `spark.read.parquet("s3a://...")` com credenciais configuradas via `spark.conf`. Isso faz o Spark ler distribuido entre os executors em vez de materializar tudo no driver, eliminando o OOM.
