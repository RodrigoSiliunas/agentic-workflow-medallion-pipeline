## Technical Notes

### Geist via @nuxt/fonts

A package `@nuxt/fonts` ja esta nas dependencias do projeto. Adicionar Geist em `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  fonts: {
    families: [
      { name: "Geist", provider: "google" },
      { name: "Geist Mono", provider: "google" },
    ],
  },
})
```

### Tokens CSS variables

Estrategia: definir variaveis em `:root` (light) e `.dark` (override), usando o padrao do `@nuxt/ui` que aceita CSS variables nos themes.

```css
:root {
  --brand-50: #F5F3FF;
  --brand-500: #8E51F6;
  --brand-600: #7F22FE;
  --brand-700: #7008E7;

  --bg: #FAFAFA;
  --surface: #FFFFFF;
  --surface-elevated: #F4F4F5;
  --border: #E4E4E7;
  --text-primary: #18181B;
  --text-secondary: #6B6B7E;
}

.dark {
  --bg: #090A0B;
  --surface: #18181B;
  --surface-elevated: #272A2E;
  --border: #2E2E3A;
  --text-primary: #F0F0F5;
  --text-secondary: #8B8B9E;
}
```
